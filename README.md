# Plants vs Zombies Fusion

A JavaFX tower-defense game inspired by Plants vs Zombies, developed using Object-Oriented Programming principles.

## Preview

This project features real-time gameplay, multiple enemy types, strategic resource management, and custom mechanics such as plant fusion and parasite zombies.

## Features

- Real-time simulation engine using JavaFX Timeline
- Multiple plant classes with unique abilities
- Multiple zombie enemy types
- Day Mode and Night Mode
- Easy / Medium / Hard difficulty levels
- Fusion mechanic for upgraded plants
- Cage trap system
- Parasite Zombie special behavior
- Win / Lose screens
- Settings menu

## Technologies Used

- Java
- JavaFX
- Object-Oriented Programming
- Git / GitHub

## Main Classes

- GameBoard – Main gameplay engine
- GameBoardView – Rendering system
- Plant – Parent class for plants
- Zombie – Parent class for zombies
- ZombieFactory – Creates zombie types

## How to Run

1. Open project in IntelliJ IDEA or VS Code
2. Configure JavaFX if needed
3. Run Main.java
